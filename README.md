# 🏋️ TopGym

TopGym é uma aplicação mobile-first para gerenciamento de academias, desenvolvida com React.

O projeto tem como objetivo oferecer uma experiência moderna para alunos, permitindo acompanhar treinos, planos, calendário, configurações e muito mais.

## ✨ Funcionalidades

- 🔐 Login
- 📝 Cadastro
- 🔑 Recuperação de senha
- 🏠 Tela inicial
- 💪 Treinos
- 📅 Calendário
- 📍 Localização
- 💳 Planos
- ⚙️ Configurações
- 🌙 Tema Claro/Escuro

## 🚀 Tecnologias

- React
- React Router
- CSS Modules
- LocalStorage
- Context API
- Firebase *(versão futura)*
- APIs REST

## 🎯 Objetivos do projeto

- Aprender React
- Criar uma aplicação mobile-first
- Consumir APIs
- Trabalhar com autenticação
- Desenvolver um projeto de portfólio
